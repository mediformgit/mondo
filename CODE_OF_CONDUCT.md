# 行動規範 — Code of Conduct

問道は、**学ぶ人すべてに開かれた場**であろうとしています。その精神を、コミュニティにも適用します。

## 約束

参加者・貢献者・運営者は、年齢・経験・所属・専門（文系/理系）・国籍・障害の有無などに関わらず、
**ハラスメントのない、敬意ある環境**を保つことに努めます。

## 推奨される態度

- 初心者の質問を歓迎する。「そんなことも知らないのか」を言わない（**入口をせばめない**は、人にも適用される）。
- 異論は、人ではなくアイデアに向ける。建設的に。
- 相手の貢献の良い点をまず認める。

## 許容されない言動

- 侮辱・差別的発言・個人攻撃・つきまとい。
- 性的な言動、他者の私的情報の無断公開。
- 議論を装った妨害・荒らし。

## 運用

問題があれば、運営者（リポジトリのメンテナ）に連絡してください。報告は誠実に扱い、報告者を保護します。
運営者は、警告・編集・削除・参加の制限など、必要な対応をとることができます。

> 知が誰にでも開かれているように、この場も、誰にでも開かれている。
