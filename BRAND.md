# ブランドと「公式サイト」について

コードは MIT、コンテンツは CC BY-SA 4.0 で**自由に使えます**。
しかし次の**ブランド要素は、いずれのライセンスにも含まれません**。フォークや派生サイトでは使用しないでください。

- 名称 **「問道」「MONDŌ」**
- ロゴ・マーク（**問**の落款印、**円相**のモチーフ）
- 公式ドメイン（運用者が `NEXT_PUBLIC_SITE_URL` に設定した本番URL）

## なぜ制限するか — なりすまし対策

このサイトは **BYOK（利用者が自分のAPIキーを持ち込む）** を提供します。
だからこそ、悪意あるフォークが「問道そっくりの見た目」で公開し、利用者を騙して鍵を盗もうとする危険があります（[PRINCIPLES.md](./PRINCIPLES.md) のメリット/デメリット参照）。

これを防ぐため：

- **フォーク・派生サイトは、別の名称・ロゴを用いること。** 「問道 / MONDŌ」を名乗らない。
- **公式サイトは1つだけ。** 利用者には、公式ドメイン以外でAPIキーを入力しないよう案内します。
- 運用者は、鍵入力UI（道場の「鍵の扱い」モーダル）や README に**公式URLを明示**することを推奨します。

## まとめ

| 要素 | 扱い | 自由に使える？ |
|---|---|---|
| コード | MIT | ✅ はい |
| コンテンツ本文 | CC BY-SA 4.0 | ✅ はい（表示＋継承の条件付き） |
| 名称・ロゴ・公式ドメイン | ブランド（権利留保） | ❌ いいえ。別名で |

善意のフォークは大歓迎です。ただ、利用者を守るために**「問道」という名前と公式の入口は、ひとつ**に保ちます。
